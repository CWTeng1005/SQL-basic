## 准备工作
1. **MySQL下载及安装**  
    下载MySQL社区版

2. **MySQL启动/关闭**  
    打开cmd输入命令行：
    > net start mysql80  
    > net stop mysql80

3. **MySQL客户端连接**  
  
    打开cmd输入命令行：  
    > mysql -u root -p  
    > 输入密码  

    <h6>注意：先配置环境</h6>
